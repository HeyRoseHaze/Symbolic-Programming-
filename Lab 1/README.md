# Lab 1 

Communication with the user, variables and basic types, operators.
## Task 1

Write a program that prints any two messages to the screen

## Task 2 

Write a program that asks the user for their name and year of birth and then calculates and prints their age.

## Task 3

Write a program that asks the user to enter two numbers a and b and prints their sum, difference, product, quotient, √(a² + b²), as well as a\^b and b\^a.

## Taks 4

Write a program that asks the user to enter the Cartesian coordinates x, y, z of a point in space, then converts them to spherical and cylindrical coordinates and prints the results.
